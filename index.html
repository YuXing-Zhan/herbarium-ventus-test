📖 系统功能详解

一、学习系统核心逻辑

Herbarium Ventus 是一款基于 Leitner 间隔重复算法的植物拉丁文学名记忆工具。系统内置 40 种植物属名的中文-拉丁文对照词库，每个词条拥有独立的六级掌握度盒子：

  • Box 0 · 新词（灰色圆点）：从未接触过的词条，首次在闪卡或测验中遇到时自动升入 Box 1
  • Box 1 · 薄弱（红色圆点）：刚接触或反复出错的词条，出现权重最高（权重 8），需要频繁复习
  • Box 2 · 初学（黄色圆点）：有一定印象但仍需巩固，权重适中（权重 4）
  • Box 3 · 熟悉（浅绿圆点）：基本掌握，权重降低（权重 2），偶尔抽查
  • Box 4 · 已掌握（绿色圆点）：稳定掌握，权重最低（权重 1），仅偶尔抽查以维持长期记忆
  • Box 5 · 归档（金色圆点）：已永久记忆，权重极低（权重 0.3），极少出现

每次在闪卡中答对或在测验中输入正确，词条盒子 +1；答错则盒子 -1。此外，标记星标（⭐）的词条权重翻倍，无论处于哪个盒子都会被系统优先推送复习。超过 24 小时未复习的 Box 3 及以下词条会额外获得 1.5 倍权重加成，确保遗忘曲线中的词条及时被重新激活。

二、三大学习模式

风之图鉴（浏览）：知识库入口，展示全部 40 种植物属名的卡片网格。每张卡片显示中文名和拉丁学名，右上角圆点直观反映当前掌握度，⭐ 表示已加重点。点击卡片选中该属后，页面下方画廊加载其实拍照片；点击 🔊 听取发音。浏览模式仅提供信息查阅，不会影响学习进度，因此不会触发盒子状态变更。

落叶闪卡（复习）：核心记忆训练模式。系统通过 Leitner 加权随机算法抽出一张卡片，正面显示中文名，点击卡片（或按空格键）翻转为拉丁文并自动朗读。用户根据实际记忆情况诚实点击「认识」「不认识」或「重点」三个按钮之一，或使用键盘 1/2/3 操作。每次判定后卡片立刻刷新为下一张推荐词条。该模式适合日常复习，薄弱词自动高频出现，已掌握词低频抽查。

飘叶测验（考试）：主动拼写检测模式。系统显示中文名，用户需在输入框中完整拼写对应的拉丁学名。输入时自动将首字母转为大写。当输入长度达到正确答案长度时自动触发判定，也可按 Enter 手动提交。答对后触发庆祝粒子特效，3 秒倒计时自动进入下一题，答错则显示正确答案并自动将该词加入错题本。测验底部实时统计正确数和正确率。支持四种出题范围：全部词条、仅薄弱词、仅错题本、仅星标词。

三、视觉与氛围系统

五季主题：点击顶部右侧五个圆点按钮切换配色方案——春·新绿（嫩绿色调）、夏·晴空（天蓝色调）、秋·红叶（暖橙色调）、冬·霜枝（冷灰蓝色调）、暮·萤火（暗夜荧光色调）。每个季节拥有完整独立的色彩系统，覆盖背景渐变、卡片底色、文字色、强调色、边框和阴影。选择自动保存，下次访问沿用。首次访问时若系统为深色模式则自动匹配「暮」主题，系统切换深色/浅色时自动跟随。

粒子生态系统：全屏 Canvas 动画渲染包含花粉微尘、落叶、花瓣、萤火虫、蒲公英种子、星火、云絮、日光束光斑共计八种粒子类型。所有粒子统一受模拟风向影响——鼠标在屏幕左右位置移动可改变风力方向和强度。萤火虫轻微被光标吸引，近距萤火虫之间产生微光连接线。随季节切换，粒子颜色自动适配当前主题调色板。

自定义光标：桌面端鼠标指针替换为叶片形状，带有柔和光晕。悬停于卡片、按钮、输入框等可交互元素上时叶片变为放大的圆形，点击时收缩模拟按压感。触屏设备自动隐藏自定义光标，使用原生触摸交互。

四、画廊与外部链接

在风之图鉴中点击任意植物卡片选中后，页面下方「植物图鉴」画廊自动加载该属的实拍照片（图片来源：PPBC 中国植物图像库），支持懒加载优化。鼠标在图片上移动时表面产生露珠折射光效。点击图片可灯箱放大查看，按 Esc 或点击遮罩关闭。选中植物后，画廊标题旁出现📖 百科链接，点击在新标签页打开该属的百度百科页面查看更多植物学资料。未选中任何植物时百科链接自动隐藏。

五、语音与音效

系统利用浏览器内置 Web Speech API 实现拉丁文发音。左上角下拉菜单选择 TTS 语音引擎，系统优先推荐意大利语女声（Elsa/Alice），因意大利语的元音发音体系最接近古典拉丁文。备选依次为德语男声、英语女声、英语男声，最后还有一个「系统默认」选项不指定语音。以下场景自动触发朗读：翻转闪卡时、测验中点击「揭示答案」时、点击任意 🔊 按钮时。朗读过程中屏幕中央扩散三道音波波纹动画提供视觉反馈。

六、键盘操作体系

所有核心操作均支持键盘快捷键，无需鼠标即可完成完整学习流程。在落叶闪卡模式下：空格翻面、1 不认识、2 切换重点、3 认识。在飘叶测验模式下：Enter 提交答案。全局快捷键：Esc 按优先级依次关闭（快捷键配置面板 → 帮助面板 → 图片灯箱）、W 返回页面顶部。点击顶部 ⌨ 按钮可打开快捷键配置面板，以可视化方式重新绑定任意按键——点击输入框后按下目标按键即完成修改，按 Esc 取消捕获，所有修改自动持久化保存。

快捷键和按钮均内置 0.4 秒冷却锁定，防止快速连按或长按导致误操作（如连续翻卡、连续提交）。窗口失去焦点时自动解除锁定，切回后可立即正常使用。

七、进度与数据管理

标题下方进度条以百分比形式直观展示整体学习进度。状态栏以颜色编码实时显示：🟢 已掌握数量、🟡 学习中的数量、🔴 薄弱词数量、⭐ 星标词数量。所有学习数据——包括每个词条的盒子等级、正确次数、错误次数、见过次数、星标状态、最后复习时间戳——全部存储在浏览器的 localStorage 中，关闭标签页、重启浏览器均不会丢失。系统提供三种重置粒度：重置全部学习进度（↺ 按钮，需要弹窗确认）、仅清零测验成绩计数（飘叶测验面板中的「重置成绩」）、恢复快捷键默认配置（快捷键配置面板中的「恢复默认」）。

八、辅助与提示

首次访问网站时帮助面板自动弹出，之后可随时点击右上角 ? 按钮查看。页面右下角 ▲ 按钮提供一键返回顶部。画廊图片采用滚动驱动渐入动画。答对测验时的庆祝粒子爆散颜色自动匹配当前季节主题。闪卡在翻面后按钮有短暂冷却防止在看清答案前误触。

⌨ 自定义快捷键

点击右侧输入框，然后按下你想要绑定的按键即可修改。修改自动保存。
